Εντολές
Εντολή Περιγραφή
printf() Εκτυπώνει κείμενο ή τιμές στην κονσόλα.
scanf() Διαβάζει τιμές από τον χρήστη από την κονσόλα.
if Ελέγχει μια συνθήκη και εκτελεί κώδικα αν η συνθήκη είναι αληθής.
else Εκτελεί κώδικα αν η συνθήκη στην if δεν είναι αληθής.
for Επαναλαμβάνει έναν κομμάτι κώδικα για έναν ορισμένο αριθμό επαναλήψεων.
while Επαναλαμβάνει έναν κομμάτι κώδικα όσο μια συνθήκη είναι αληθής.
do-while Εκτελεί ένα κομμάτι κώδικα τουλάχιστον μία φορά και μετά επαναλαμβάνει τον κώδικα όσο μια συνθήκη είναι αληθής.
switch Εκτελεί διαφορετικές ενέργειες ανάλογα με την τιμή μιας μεταβλητής.
break Τερματίζει την εκτέλεση ενός βρόχου ή ενός switch statement.
continue Παρακάμπτει τον υπόλοιπο κώδικα ενός βρόχου και συνεχίζει με την επόμενη επανάληψη.
return Επιστρέφει μια τιμή από μια συνάρτηση.
