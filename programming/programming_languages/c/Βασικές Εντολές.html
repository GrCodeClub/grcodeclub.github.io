Βασικές Εντολές
Εντολή Περιγραφή
printf() Εκτυπώνει κείμενο ή τιμές στην κονσόλα.
scanf() Διαβάζει τιμές από τον χρήστη από την κονσόλα.
if Ελέγχει μια συνθήκη και εκτελεί κώδικα αν η συνθήκη είναι αληθής.
else Εκτελεί κώδικα αν η συνθήκη στην if δεν είναι αληθής.
for Επαναλαμβάνει έναν κομμάτι κώδικα για έναν ορισμένο αριθμό επαναλήψεων.
while Επαναλαμβάνει έναν κομμάτι κώδικα όσο μια συνθήκη είναι αληθής.
do-while Εκτελεί ένα κομμάτι κώδικα τουλάχιστον μία φορά και μετά επαναλαμβάνει τον κώδικα όσο μια συνθήκη είναι αληθής.
switch Εκτελεί διαφορετικές ενέργειες ανάλογα με την τιμή μιας μεταβλητής.
break Τερματίζει την εκτέλεση ενός βρόχου ή ενός switch statement.
continue Παρακάμπτει τον υπόλοιπο κώδικα ενός βρόχου και συνεχίζει με την επόμενη επανάληψη.
return Επιστρέφει μια τιμή από μια συνάρτηση.

Βασικές βιβλιοθήκες

Βιβλιοθήκη Περιγραφή
#include <stdio.h> Βιβλιοθήκη εισόδου/εξόδου που παρέχει συναρτήσεις όπως printf() και scanf().
#include <stdlib.h> Βιβλιοθήκη που παρέχει συναρτήσεις για τη διαχείριση μνήμης, δυναμική ανάθεση μνήμης και άλλες γενικές λειτουργίες.

#include <string.h> Βιβλιοθήκη για λειτουργίες σχετικές με τις συμβολοσειρές, όπως αντιγραφή, συγκατάθεση, σύγκριση και άλλες.

#include <stdbool.h> Βιβλιοθήκη που παρέχει τον τύπο δεδομένων bool και τις τιμές true και false για λογικές εκφράσεις.

Εντολές if, if-else, else


Οι εντολές if, if-else και else χρησιμοποιούνται για τον έλεγχο μιας συνθήκης και την εκτέλεση διαφορετικών τμημάτων κώδικα ανάλογα με την αποτίμηση της συνθήκης.

Η σύνταξη των εντολών είναι η εξής:


            if (condition) {
            // Κώδικας που εκτελείται αν η συνθήκη είναι αληθής
            }
            else {
            // Κώδικας που εκτελείται αν η συνθήκη είναι ψευδής
            }

Με την εντολή `if`, ελέγχεται μια συνθήκη και εάν είναι αληθής, εκτελείται ο κώδικας μέσα στο μπλοκ που ακολουθεί.Αν η συνθήκη είναι ψευδής, ο κώδικας παραλείπεται και συνεχίζεται η εκτέλεση των υπόλοιπων τμημάτων κώδικα. Με την εντολή `if-else`, εκτελείται ένα μπλοκ κώδικα αν η συνθήκη είναι αληθής και ένα διαφορετικό μπλοκ κώδικα αν η συνθήκη είναι ψευδής. Τέλος, η εντολή `else` χρησιμοποιείται για την εκτέλεση ενός μπλοκ κώδικα όταν καμία από τις προηγούμενες συνθήκες δεν είναι αληθείς.



Εντολή switch


Η εντολή switch χρησιμοποιείται για τον έλεγχο μιας έκφρασης (expression) και την εκτέλεση διαφορετικών τμημάτων κώδικα ανάλογα με την τιμή της έκφρασης.
Η σύνταξη της εντολής switch είναι η εξής:

            switch (expression) {
      case value1:
        // Κώδικας που εκτελείται αν η έκφραση είναι ίση με την value1
        break;
      case value2:
        // Κώδικας που εκτελείται αν η έκφραση είναι ίση με την value2
        break;
      default:
        // Κώδικας που εκτελείται αν η έκφραση δεν ταιριάζει με καμία από τις προηγούμενες τιμές
    }

Η εντολή switch ελέγχει την τιμή της έκφρασης και εκτελεί τον κώδικα ανάλογα με την ταιριάζουσα τιμή. Κάθε τιμή ορίζεται με την εντολή case και τον αντίστοιχο κώδικα που θα εκτελεστεί. Ο τερματισμός του κάθε τμήματος κώδικα γίνεται με την εντολή break.Αν καμία τιμή δεν ταιριάζει, μπορεί να χρησιμοποιηθεί η εντολή default για την εκτέλεση ενός προαιρετικού τμήματος κώδικα.
