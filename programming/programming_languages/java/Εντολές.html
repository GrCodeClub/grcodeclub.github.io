Εντολή Περιγραφή
System.out.println("Hello, World!"); Εμφανίζει το κείμενο "Hello, World!" στην κονσόλα.
if (x > 0) {
  System.out.println("Το x είναι θετικό");
}		 Ελέγχει αν η τιμή της μεταβλητής x είναι μεγαλύτερη από το 0 και εμφανίζει το αντίστοιχο μήνυμα.
for (int i = 0; i < 10; i++) {
  System.out.println(i);
}		 Επαναλαμβάνει τον κώδικα μέχρι την εκτέλεση του συνθήκης i < 10, αυξάνοντας την τιμή της μεταβλητής i κατά 1 σε κάθε επανάληψη και εμφανίζει την τρέχουσα τιμή της i.
while (x > 0) {
  System.out.println(x);
  x--;
}		 Επαναλαμβάνει τον κώδικα μέχρι την εκτέλεση του συνθήκης x > 0, μειώνοντας την τιμή της μεταβλητής x κατά 1 σε κάθε επανάληψη και εμφανίζει την τρέχουσα τιμή της x.
