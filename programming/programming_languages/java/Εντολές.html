static final - Χρησιμοποιείται για τον ορισμό σταθερών τιμών. Μια σταθερά τιμή δεν μπορεί να αλλάξει μετά την αρχικοποίησή της.
instanceof Χρησιμοποιείται για τον έλεγχο του τύπου ενός αντικειμένου.
hashCode() Μέθοδος που χρησιμοποιείται για την ανάκτηση του hash code ενός αντικειμένου.
Παράδειγμα:
int hashCode = obj.hashCode();

Σύγκριση

equals() Μέθοδος που χρησιμοποιείται για τη σύγκριση της ισότητας δύο αντικειμένων.
Παράδειγμα:
if (obj1.equals(obj2)) { }
compareTo() Μέθοδος που χρησιμοποιείται για τη σύγκριση της σειράς δύο αντικειμένων.
Παράδειγμα:
int result = obj1.compareTo(obj2);
compareToIgnoreCase() Μέθοδος που χρησιμοποιείται για τη σύγκριση της σειράς δύο αλφαριθμητικών αγνοώντας την πεζή/κεφαλαία γραφή τους.
Παράδειγμα:
int result = str1.compareToIgnoreCase(str2);
compare() Μέθοδος που χρησιμοποιείται για τη σύγκριση δύο αντικειμένων βάσει μιας προσαρμοσμένης συνάρτησης σύγκρισης.
Συνήθως χρησιμοποιείται σε συνδυασμό με το Comparator interface
Παράδειγμα:
int result = comparator.compare(obj1, obj2);
