Συμβολοσειρές
Εντολή Περιγραφή

Μετατροπή σε άλλους τύπους (stdlib.h)
atoi Μετατρέπει μια συμβολοσειρά σε ακέραιο.
atol Μετατρέπει μια συμβολοσειρά σε ακέραιο μεγάλου μεγέθους.
atof Μετατρέπει μια συμβολοσειρά σε δεκαδικό αριθμό.

Συναρτήσεις διαχείρισης συμβολοσειρών (string.h)
char* strcat (char* dest, const char* src); Παραθέτει τη συμβολοσειρά src στο τέλος της dest. Επιστρέφει την dest.
int strcmp (const char* string1, const char* string2); Συγκρίνει τις συμβολοσειρές string1 and string2. Επιστρέφει 0 όταν είναι ίσες.
char* strcpy (char* dest, const char* src); Αντιγράφει το περιεχόμενο της src στη dest. Επιστρέφει την dest.
size_t strlen (const char* string); Επιστρέφει το μήκος της συμβολοσειράς.
