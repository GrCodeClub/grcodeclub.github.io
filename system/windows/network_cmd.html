    Εντολές Δικτύου

    ipconfig

    Εμφανίζει πληροφορίες δικτύου για όλες τις συνδεδεμένες δικτυακές συσκευές.

    ipconfig

    Ανανεώνει την IP διεύθυνση για μια δικτυακή συσκευή από τον DHCP διακομιστή.

    ipconfig /renew

    Απελευθερώνει την IP διεύθυνση που έχει αντιστοιχιστεί σε μια δικτυακή συσκευή.

    ipconfig /release

    ping

    Στέλνει ένα αίτημα ICMP echo σε έναν προορισμό για να ελέγξει τη συνδεσιμότητα και την απόκριση.

    ping

    tracert

    Ανιχνεύει τη διαδρομή που ακολουθεί ένα πακέτο μέσα στο δίκτυο προς έναν προορισμό.

    tracert

    netstat

    Εμφανίζει τις στατιστικές πληροφορίες για τις δικτυακές συνδέσεις, τις θύρες και τους πίνακες δρομολόγησης.

    netstat

    nslookup

    Εκτελεί αναζήτηση DNS για να αντιστοιχίσει ένα όνομα κεντρικού υπολογιστή σε μια διεύθυνση IP.

    nslookup

    arp

    Εμφανίζει και τροποποιεί τον πίνακα ARP (Address Resolution Protocol) που αντιστοιχεί διευθύνσεις IP σε φυσικές διευθύνσεις MAC.

    arp

    netsh

    Εκτελεί εργασίες ρύθμισης δικτύου, όπως ρύθμιση διευθύνσεων IP, θυρών, δρομολογητών κ.λπ.

    netsh

    route

    Εμφανίζει ή τροποποιεί τους πίνακες δρομολόγησης IP.

    route

    troubleshooter

    Εκτελεί το εργαλείο επίλυσης προβλημάτων δικτύου των Windows για αυτόματη διάγνωση και επιδιόρθωση προβλημάτων.

    troubleshooter

    netstat

    Εμφανίζει πληροφορίες για τις ενεργές συνδέσεις δικτύου, τις θύρες που είναι ανοιχτές και άλλα στατιστικά δεδομένα.

    netstat

    Εμφανίζει όλες τις ενεργές συνδέσεις δικτύου, συμπεριλαμβανομένων των αριθμών θύρας και των διευθύνσεων IP.

    netstat -an
