    Εντολές Δικτύου


    Εντολή Περιγραφή
    ipconfig Εμφανίζει τις πληροφορίες δικτύου για όλες τις συνδεδεμένες δικτυακές συσκευές.
    ipconfig /release Απελευθερώνει την IP διεύθυνση που έχει αντιστοιχιστεί σε μια δικτυακή συσκευή.
    ipconfig /renew Ανανεώνει την IP διεύθυνση για μια δικτυακή συσκευή από τον DHCP διακομιστή.
    ping Στέλνει ένα αίτημα ICMP echo σε έναν προορισμό για να ελέγξει τη συνδεσιμότητα και την απόκριση.
    tracert Ανιχνεύει τη διαδρομή που ακολουθεί ένα πακέτο μέσα στο δίκτυο προς έναν προορισμό.
    netstat Εμφανίζει τις στατιστικές πληροφορίες για τις δικτυακές συνδέσεις, τις θύρες και τους πίνακες δρομολόγησης.
    nslookup Εκτελεί αναζήτηση DNS για να αντιστοιχίσει ένα όνομα κεντρικού υπολογιστή σε μια διεύθυνση IP.
    arp Εμφανίζει και τροποποιεί τον πίνακα ARP (Address Resolution Protocol) που αντιστοιχεί διευθύνσεις IP σε φυσικές διευθύνσεις MAC.
    netsh Εκτελεί εργασίες ρύθμισης δικτύου, όπως ρύθμιση διευθύνσεων IP, θυρών, δρομολογητών κ.λπ.
    nbtstat Εμφανίζει στατιστικά και πληροφορίες NetBIOS (Network Basic Input/Output System).
    route Εμφανίζει ή τροποποιεί τους πίνακες δρομολόγησης IP.
    troubleshooter Εκτελεί το εργαλείο επίλυσης προβλημάτων δικτύου των Windows για αυτόματη διάγνωση και επιδιόρθωση προβλημάτων.
    netstat -an Εμφανίζει όλες τις ενεργές συνδέσεις δικτύου, συμπεριλαμβανομένων των αριθμών θύρας και των διευθύνσεων IP.
    netstat Εμφανίζει πληροφορίες για τις ενεργές συνδέσεις δικτύου, τις θύρες που είναι ανοιχτές και άλλα στατιστικά δεδομένα.
