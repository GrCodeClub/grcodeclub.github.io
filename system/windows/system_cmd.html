Εντολές Συστήματος


Εντολή Περιγραφή
systeminfo Εμφανίζει λεπτομερείς πληροφορίες για το σύστημα, συμπεριλαμβανομένων των πληροφοριών του λειτουργικού συστήματος, του επεξεργαστή, της μνήμης και άλλων στοιχείων.
tasklist Εμφανίζει τη λίστα των εκτελούμενων διεργασιών στο σύστημα, μαζί με λεπτομέρειες όπως το ID διεργασίας, την κατάσταση και την κατανάλωση πόρων.
taskkill Τερματίζει μια εκτελούμενη διεργασία βάσει του ID διεργασίας ή του ονόματος.
msconfig Εμφανίζει το παράθυρο διαμόρφωσης του συστήματος, όπου μπορείτε να διαμορφώσετε τις ρυθμίσεις εκκίνησης του συστήματος.
chkdsk Ελέγχει την κατάσταση του σκληρού δίσκου και επιδιορθώνει τυχόν σφάλματα αρχείων.
sfc /scannow Εκτελεί μια επισκευή συστήματος για να εντοπίσει και να επιδιορθώσει τυχόν αρχεία συστήματος που έχουν καταστραφεί.
gpedit.msc Ανοίγει το παράθυρο διαμόρφωσης πολιτικής ομάδας του συστήματος, όπου μπορείτε να διαμορφώσετε τις πολιτικές του συστήματος.
shutdown Τερματίζει το σύστημα ή τον υπολογιστή.
taskmgr Ανοίγει τον Task Manager, ένα εργαλείο που σας επιτρέπει να διαχειρίζεστε τις εργασίες και τις διεργασίες που εκτελούνται στον υπολογιστή σας
control panel Ανοίγει το Control Panel, όπου μπορείτε να διαχειριστείτε διάφορες ρυθμίσεις και επιλογές του συστήματος, όπως τη διαχείριση συσκευών, την απεγκατάσταση προγραμμάτων, την αλλαγή των ρυθμίσεων του δικτύου κ.λπ.
services.msc Ανοίγει το παράθυρο Services, όπου μπορείτε να διαχειριστείτε τις υπηρεσίες που εκτελούνται στον υπολογιστή σας. Μπορείτε να ενεργοποιήσετε, απενεργοποιήσετε ή αλλάξετε τις ρυθμίσεις των υπηρεσιών.
compmgmt.msc Ανοίγει το παράθυρο Computer Management, όπου μπορείτε να διαχειριστείτε διάφορες πτυχές του συστήματος, όπως τους σκληρούς δίσκους, τις συσκευές, τις κοινόχρηστες φακέλους, τον χώρο αποθήκευσης κ.λπ.
regedit Ανοίγει το Registry Editor, που σας επιτρέπει να τροποποιήσετε το Μητρώο των Windows, όπου αποθηκεύονται οι ρυθμίσεις και οι παράμετροι του συστήματος.
wmic process Εκτελεί το Windows Management Instrumentation Command-line (WMIC) και παρέχει προηγμένες λειτουργίες διαχείρισης των διεργασιών, όπως η εμφάνιση λεπτομερών πληροφοριών και η διαχείριση των προτεραιοτήτων των διεργασιών.
diskpart Εκκινεί το DiskPart, ένα εργαλείο γραμμής εντολών που σας επιτρέπει να διαχειρίζεστε τους σκληρούς δίσκους, τις κατατμήσεις και τους τόμους. Μπορείτε να δημιουργήσετε, να διαγράψετε, να μεταβάλετε το μέγεθος και να διαμορφώσετε κατατμήσεις.
defrag κτελεί τον εργαλείο αποκατάστασης συστήματος αρχείων (Disk Defragmenter) για τη βελτιστοποίηση της τοποθέτησης των αρχείων στον σκληρό δίσκο.
diskmgmt.msc Ανοίγει το παράθυρο Disk Management, όπου μπορείτε να διαχειριστείτε τους δίσκους, τις κατατμήσεις και τους τόμους του συστήματος. Μπορείτε να δημιουργήσετε, να διαγράψετε, να αντιγράψετε, να μεταβάλετε το μέγεθος και να διαμορφώσετε τις κατατμήσεις.
